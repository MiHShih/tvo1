# 決定初動（上下動）

在確定波相到時的同時，可以同時決定波相得初動。初動可以在多波形和單一波形模式下選取，但在單一波形模式下最為簡單。

下圖是單一波形模式下的放大波形，因此上下動清晰可見。對於要決定的初動，滑鼠游標必須高於或低於**'Limits for polarity**'標記。如下圖所示放置游標，按'1'，結果如圖所示。如果以前沒有初動的讀數，只需重新選取一次即可。
